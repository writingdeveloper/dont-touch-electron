# Не трогай

**ИИ-приложение для преодоления привычек вроде трихотилломании**

[English](../README.md) | [한국어](README.ko.md) | [日本語](README.ja.md) | [中文](README.zh.md) | [Español](README.es.md)

---

## Описание

Настольное приложение, использующее веб-камеру для обнаружения в реальном времени приближения руки к лицу. Помогает прервать повторяющееся поведение, такое как **трихотилломания** (выдёргивание волос) и **дерматилломания** (ковыряние кожи), с помощью визуальных и звуковых оповещений.

Вся обработка происходит локально на вашем устройстве. Данные не собираются и не передаются.

## Функции

- Обнаружение лица и рук в реальном времени через MediaPipe
- Настраиваемые зоны обнаружения (кожа головы, брови, глаза, щёки и т.д.)
- Полноэкранные оповещения + звук предупреждения
- Ежедневная статистика и отслеживание серий
- Встроенная дыхательная медитация
- Поддержка системного трея
- Многоязычный интерфейс

## Установка

### Загрузка

Скачайте последнюю версию со страницы [Releases](https://github.com/writingdeveloper/dont-touch-electron/releases).

### Сборка из исходников

```bash
git clone https://github.com/writingdeveloper/dont-touch-electron.git
cd dont-touch-electron
npm install
npm run dev      # Режим разработки
npm run build    # Production сборка
```
